#### Git设置github邮箱地址和用户名
* 在根目录下执行 Git Bash窗口中分别执行如下代码
    ```
        // 设置邮箱
        $ git config --global user.email "xxx.com"
    ```
    ```
        // 设置名称
        git config --global user.name "Axxxt"
    ```
